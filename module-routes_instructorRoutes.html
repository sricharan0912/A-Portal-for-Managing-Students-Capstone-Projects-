Instructor Routes Module

Handles all instructor-related API endpoints including authentication, profile management, course settings, student management, group formation, and dashboard statistics.

Routes are organized by functionality:

  1. Authentication (signup, login)
  2. Settings Management (preference deadline, course configuration)
  3. Student Management (add students, view all students)
  4. Group Formation (auto-assign, preview, manual management)
  5. Instructor Profile (CRUD operations)
  6. Dashboard Statistics

IMPORTANT: Group-related routes MUST be defined BEFORE /:instructor_id routes to prevent Express from treating "groups" as an instructor ID parameter.
