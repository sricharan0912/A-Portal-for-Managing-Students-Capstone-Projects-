(async, inner) getInstructorStats(req, res) → {Promise.<void>}

Aggregates platform-wide statistics across three main categories:

Students:

  • Total active student count

Projects:

  • Total project count
  • Count by status (open, approved, closed)

Groups:

  • Total groups created

Note: Statistics are platform-wide, not instructor-specific Only includes active (non-deleted) users
