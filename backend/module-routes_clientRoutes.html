Module: routes/clientRoutes

Client Routes Module

Handles all client-related API endpoints including authentication, profile management, project management, and team/group assignments.

Routes are organized by functionality:

  1. Authentication (signup, login)
  2. Client Profile (CRUD operations)
  3. Client Projects (list, statistics)
  4. Teams/Groups (assigned to client's projects)

Generate JWT Token for Client

Creates a signed JWT token containing client authentication data. Token expires in 7 days.
